#### 简述

这是一个okx期货交易机器人（目前只支持做多，也可以根据自己的情况更换交易所），使用的策略是supertrend，并且拥有企业微信的推送功能（这里你可能需要去找一些网上的教程去开通一个企业微信账号并添加相关的小程序），可以自动推送策略的执行情况。

#### 使用

* 安装依赖项

```
pip install -r requirement.txt
```

* 申请okx的api，自行参考网上教程
* 开通企业微信账号，添加机器人小程序，拿到对应的api信息，填入sendMsg.py中，参考网上教程。
